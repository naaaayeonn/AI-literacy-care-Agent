# 3번 피드백 반영 완료 및 연동 가이드 (RESPONSE_TO_ROLE3_FEEDBACK)

안녕하세요 3번(Cognitive Care Backend) 담당자님! 전달해주신 피드백(`3)peedback.txt`) 내용을 바탕으로 **RAG 및 단어 조회 엔진 고도화**를 모두 완료하고 깃허브 원격 저장소에 반영했습니다. 연동 시 알아두셔야 할 유의사항 및 변경 정보를 정리해 드립니다.

---

## 1. 피드백 반영 완료 사항 (Action Items)

### [Step 1] 동적 LLM 실시간 의미 유추 구현
- 로컬 사전, 우리말샘 API, 임베딩 유사성 매칭 모두에서 단어의 뜻을 찾지 못할 경우, **Gemini 2.0 Flash를 활용해 실시간으로 의미를 유추**하여 제공합니다.
- 사용자가 준 `context`(기사 문맥)를 고려하여 **50자 이내**로 친절하게 설명하며, 응답 시 `source="LLM 실시간 유추"`로 표기됩니다.

### [Step 2] 공인 출처 2차 탐색 (우리말샘 API 연동)
- 공공데이터포털의 국립국어원 '우리말샘 오픈 API' 연동이 정상 작동합니다. 환각이 없는 공신력 있는 국어사전 정의를 실시간으로 검색해 옵니다.

### [Step 3] 한국어 조사 및 문장부호 전처리 파이프라인 추가
- 사용자가 조사가 포함된 구절(예: `"방지법을"`, `"메타인지는"`)을 드래그하여 보내더라도, 특수문자와 조사("~을", "~를", "~은", "~는", "~이", "~가", "~의", "~에" 등)를 지능적으로 깎아내어 **명사 원형을 도출**합니다.
- 탐색 시 **[원본 드래그어 ➡️ 조사 제거 원형]** 순서로 매칭을 교차 시도하여 검색 성공률을 극대화했습니다.

## 2. 3번 백엔드 연동 시 핵심 유의사항 (중요)

### 💡 `context` 필드 전송 (매우 권장)
- `/api/terms/lookup` 엔드포인트를 호출할 때, 사용자가 드래그한 **단어가 등장한 문장 또는 문단**을 `context` 필드에 실어 전달해 주세요.
- 문맥(`context`) 정보가 제공되어야 LLM이 실시간 유추를 할 때 기사 내용과 100% 일치하는 알맞은 동적 뜻풀이(예: 동음이의어 구분 등)를 생성할 수 있습니다.

### 🔑 `.env` 환경 변수 구성
- 실시간 API가 올바르게 호출되려면 본인 로컬의 `.env` 파일에 아래 키가 설정되어 있어야 합니다:
  - `GEMINI_API_KEY`: Gemini API 키 (없을 경우 LLM 유추가 작동하지 않고 조용히 미발견 폴백)
  - `WOORIMAL_API_KEY`: 우리말샘 API 키 (없을 경우 우리말샘 실시간 조회가 누락되고 임베딩 유사도로 폴백)

## 3. 단어 조회 API 명세 및 규격

- **Endpoint**: `POST /api/terms/lookup`
- **Request Body 규격:**
  ```json
  {
    "word": "방지법을",
    "sessionId": "sess_test123", // (선택)
    "context": "이번에 통과된 주가 누르기 방지법을 위반할 경우 처벌받습니다." // (문맥 유추용, 매우 권장)
  }
  ```

- **Response Body 규격 (snake_case/camelCase 이중 호환 완료):**
  ```json
  {
    "term": "방지법", // 조사가 깎인 원형 단어 반환
    "definition": "어떠한 행위나 현상을 미연에 막기 위한 법률.", // 실시간 정의
    "source": "우리말샘 (국립국어원)", // 아래 출처 구분 참고
    "faithfulnessScore": 1.0,
    "faithfulness_score": 1.0,
    "chunkId": "",
    "chunk_id": ""
  }
  ```

### ※ 출처(`source`) 구분값 정리
3번 백엔드 및 4번 프론트엔드에서 데이터 출처 구분에 활용할 수 있습니다.
- `"로컬 사전"`: `data/term_dictionary.json`에 정의된 106개 고정 단어장 매칭
- `"우리말샘 (국립국어원)"`: 우리말샘 오픈 API 실시간 조회 성공
- `"RAG 유사 매칭"`: 로컬 단어장에서 유사어 검색 매칭 성공 (임베딩 유사도 0.3 이상)
- `"LLM 실시간 유추"`: 신규 추가된 Gemini 2.0 Flash 실시간 의미 유추 성공
- `"not_found"`: 전체 실패 시 조용히 무시 가능하도록 가드
