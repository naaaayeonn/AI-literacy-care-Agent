Multi-Agent Architecture

AI Literacy Care Agent

글을 대신 읽어주는 AI가 아니라, 사람이 읽는 힘을 잃지 않도록 여러 에이전트가 한 세션을 함께 관리하는 구조.

1. Control Tower

Orchestrator

상태, 흐름, 점수, 자기검증을 한 곳에서 조율하고 모든 에이전트의 결과를 하나의 세션 기록으로 묶는다.

terms / chunks reading events intervention warnings

2. Text Intelligence

Content Reducer & RAG

원문을 읽기 가능한 단위로 나누고, 어려운 용어와 핵심 문장을 정리한다.

  • 쉬운 문장 생성
  • 용어 풀이
  • 문단 청크화

3. Behavior Intelligence

Cognitive Care Backend

스크롤, 멈춤, 이탈 같은 읽기 행동을 분석해 집중도와 개입 필요성을 추론한다.

  • 읽기 행동 수집
  • focus score 계산
  • 실시간 개입 신호

4. Experience Layer

Frontend & Extension

브라우저에서 사용자의 읽기 흐름을 측정하고 필요한 순간에 시각적 피드백을 보여준다.

  • 하이라이트
  • 넛지 메시지
  • 퀴즈 카드

5. Trust Intelligence

QA & Evaluation

결과의 품질, 재현성, 이상 징후를 검증해 시스템이 스스로 흔들리지 않게 만든다.

  • 품질 경고
  • 환각 방지
  • 재현성 검증

Closed Loop Literacy Care

문서를 분석하고, 행동을 관찰하고, 필요한 순간 개입하며, 마지막에는 이해도와 성장 추세를 다시 다음 세션의 개인화 신호로 되돌린다.

Measure 브라우저 안에서 읽기 행동과 문서 난이도를 측정한다.
Intervene 집중이 흔들리는 순간 하이라이트, 넛지, 퀴즈로 개입한다.
Grow 세션 결과를 점수와 프로필로 남겨 다음 읽기를 더 개인화한다.
